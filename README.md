# 🖥️ Screen Line Fix (Software Workaround)

After I noticed a **black horizontal line at the bottom of my laptop screen**.  
After trying multiple “solutions” online with no real success, I decided to take a different approach.

Instead of fixing the hardware, I built a **simple software workaround**.

This project creates a thin, always-on-top overlay that hides damaged areas of your display — clean, lightweight, and effective.

---

## ✨ Features

- 🧱 Borderless overlay (no window frame)
- 📌 Always stays on top
- 🎚️ Adjustable height
- 🔍 Optional transparency (opacity control)
- 🖱️ Click-through mode (doesn’t block mouse input)
- 📍 Position control (top or bottom)
- ⚡ Lightweight and fast

---

## 🚀 Installation

Clone the repository:

git clone https://github.com/your-username/Display-Line-fixer.git

cd screen-line-fix


Install dependencies:
pip install -r requirements.txt

Run the app:
python main.py
